cdist-type__chroot_umount(7)
============================

NAME
----
cdist-type__chroot_umount - unmount a chroot mounted by __chroot_mount


DESCRIPTION
-----------
Undo what __chroot_mount did.


REQUIRED PARAMETERS
-------------------
None


OPTIONAL PARAMETERS
-------------------
None


EXAMPLES
--------

.. code-block:: sh

    __chroot_umount /path/to/chroot


SEE ALSO
--------
:strong:`cdist-type__chroot_mount`\ (7)
